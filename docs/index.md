# excella shorts-cutter

AI-инструмент, который превращает длинные видео в готовые shorts: вырезает
лучшие моменты, делает вертикальный кроп с face-tracking, добавляет субтитры,
озвучку и брендинг.

## Что это

- **Self-hosted, free forever.** Скачали — поставили — пользуетесь.
- **Локальная обработка.** Видео не уходит в облако, всё на вашей машине.
- **Полный pipeline.** YouTube/файл → транскрипция → AI-выбор моментов →
  smart-reframe в 9:16 → субтитры → бренд → готовые мp4.
- **Web UI + REST API.** Удобно для ручного запуска и интеграции.

## Что нужно от железа

| Конфигурация | Минимум | Рекомендуется |
|---|---|---|
| RAM | 8 GB | 16 GB |
| CPU | 4 ядра | 8+ ядер |
| Диск | 20 GB free | 50+ GB |
| GPU | не обязательно | Apple M1/M2/M3/M4 (MPS) или NVIDIA с CUDA |
| Интернет | для скачивания YouTube | для регистрации лицензии |

На M4 16 GB реалистичный поток: **5–8 видео/час** (одно 30-минутное видео ≈
7–10 минут обработки в parallel-режиме).

## Что под капотом

- **Whisper** (mlx-whisper на Apple Silicon, faster-whisper на Linux/Windows)
  для транскрипции
- **YOLO** (v26/v11/v8) для детекции людей и экранов
- **MediaPipe Face Detector** + IoU-tracking для лиц
- **LR-ASD** для определения активного спикера
- **ffmpeg** для всего видео-рендера
