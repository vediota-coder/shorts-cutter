Как получить токен (требует Business аккаунт)
  1. Instagram → переключи аккаунт на Бизнес или Автор
  2. Свяжи с Facebook Page
  3. Создай приложение на developers.facebook.com/apps → добавь продукт «Instagram Graph API»
  4. В Graph API Explorer получи User Token со scope instagram_basic, instagram_content_publish, pages_read_engagement, pages_show_list, business_management
  5. Обменяй на long-lived (живёт ~60 дней) через /oauth/access_token?grant_type=fb_exchange_token
  6. Получи IG User ID: /me/accounts → найди page → /{page-id}?fields=instagram_business_account
⚠ IG требует ПУБЛИЧНЫЙ URL для видео. На локалке нужен туннель — cloudflared или ngrok.

🎙 ElevenLabs API (озвучка/дубляж)

Ключ для синтеза речи и Dubbing API. Получить: elevenlabs.io/app/settings/api-keys. Free план — TTS до 10к симв/мес. Для clone-дубляжа (голос оригинала) нужен Pro ($99/мес).

Как получить ключ (3 шага)
  1. Зарегистрируйся на elevenlabs.io (можно через Google).
  2. Открой Settings → API Keys и нажми Create API Key. Имя любое — например shorts-cutter.
  3. Скопируй строку sk_… и вставь сюда ⬇

Хранится локально в .env, никуда не уходит. Кнопка «Проверить» делает probe-запрос к API и показывает твой план + доступ к Dubbing API.

HuggingFace токен

Нужен, чтобы быстро скачивать модели Whisper. Без токена скорость ~1 MB/s, с токеном — 10+ MB/s. Бесплатно, занимает 1 минуту.

Как получить токен (3 шага)
  1. Зарегистрируйся (или войди) на huggingface.co. Можно через GitHub/Google.
  2. Открой huggingface.co/settings/tokens и нажми «+ Create new token».
  3. Тип токена: Read (этого достаточно). Имя любое — например shorts-cutter. Нажми «Create token», скопируй строку hf_… и вставь сюда ⬇

Токен хранится только локально (в файле .env с правами 600), никуда не уходит.
